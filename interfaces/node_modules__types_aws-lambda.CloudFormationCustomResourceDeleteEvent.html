    PhysicalResourceId: string

    A required custom resource provider-defined physical ID that is unique for that provider. The value returned for a PhysicalResourceId can change custom resource update operations. If the value returned is the same, it is considered a normal update. If the value returned is different, AWS CloudFormation recognizes the update as a replacement and sends a delete request to the old resource.
